Функция создает буффер заданный длины со случайными значениями.

Функция создает новый буффер заданного размера size и заполняет его псевдо-случайными значениями, выработанными с помощью заданного генератора generator.

Аргументы
generatorГенератор псевдослучайных чисел, используемый для выработки случайного заполнения буффера
sizeРазмер создаваемого буффера в байтах.
Возвращает
Функция возвращает указатель на созданный буффер. В случае возникновения ошибки возвращается NULL. Код ошибки может быть получен с помощью вызова ak_error_get_value().

int ak_buffer_wipe ( ak_buffer  buff,
ak_random  generator 
)

Зачистка данных, хранящихся в буффере.

Функция заполняет буффер случайными данными, выработанными заданным генератором псевдослучайных чисел. При этом, новые (случайные) данные не уничтожаются.

Аргументы
buffБуффер, данные которого уничтожаются
generatorГенератор псевдо-случайных чисел, используемый для генерации случайного мусора.
Возвращает
Функция возвращает ak_error_ok в случае успешного уничтожения данных. В противном случае возвращается код ошибки.

Зачистка данных, на которые указывает заданный указатель.

Функция заполняет заданную область памяти случайными данными, выработанными заданным генератором псевдослучайных чисел.

Аргументы
ptrОбласть данных, которая заполняется случайным мусором.
sizeРазмер заполняемой области в байтах.
generatorГенератор псевдо-случайных чисел, используемый для генерации случайного мусора.
Возвращает
Функция возвращает ak_error_ok в случае успешного уничтожения данных. В противном случае возвращается код ошибки.

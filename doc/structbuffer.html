Подробное описание

Класс для хранения двоичных данных

Класс рассматривается как хранилище данных, для которых контролируется размер и функции выделения/освобождения памяти. Класс может использоваться для хранения строк.

Примеры:
example-context-manager.c, example-hash-oids.c, example-hash.c, example-internal-hash01.c, example-internal-hash03.c, example-random-system.c и example-random.c.

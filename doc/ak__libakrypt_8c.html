int ak_libakrypt_create ( ak_function_log *  logger)

Функция инициализации и тестирования криптографических механизмов библиотеки.

Функция должна вызываться перед использованием криптографических механизмов библиотеки.

Пример использования функции.

1  int main( void )
2 {
3  if( ak_libakrypt_create( NULL ) != ak_true ) {
4  // инициализация выполнена не успешна => выход из программы
5  return ak_libakrypt_destroy();
6  }
7 
8  // ... здесь код программы ...
9 
10  return ak_libakrypt_destroy();
11 }
Аргументы
loggerУказатель на функцию аудита. Может быть равен NULL.
Возвращает
Функция возвращает ak_true (истина) в случае успешной инициализации и тестирования библиотеки. В противном случае, возвращается ak_false. Код ошибки может быть получен с помощью вызова функции ak_error_get_value()
Примеры:
example-bckey-internal.c, example-context-manager-node.c, example-context-manager.c, example-hash-oids.c, example-hash.c, example-internal-compress01.c, example-internal-hash02.c, example-internal-hash03.c, example-log.c, example-mpzn-convert.c, example-mpzn-montgomery.c, example-mpzn-rem.c, example-oid.c, example-random-system.c, example-random.c и example-randomize.c.

Граф вызова функции:

int ak_log_set_level ( int  level)

Установка уровня аудита библиотеки.

Неэкспортируемая функция установления уровня аудита.

Все сообщения библиотеки могут быть разделены на три уровня.

  • Первый уровень аудита определяется константой ak_log_none. На этом уровне выводятся сообщения об ошибках, а также минимальный набор сообщений, включающий в себя факт успешного тестирования работоспособности криптографических механизмов.
  • Второй уровень аудита определяется константой ak_log_standard. На этом уровене выводятся все сообщения из первого уровня, а также сообщения о фактах использования ключевой информации.
  • Третий (максимальный) уровень аудита определяется константой ak_log_maximum. На этом уровне выводятся все сообщения, доступные на первых двух уровнях, а также сообщения отладочного характера, позхволяющие прослдедить логику работы функций библиотеки.
Аргументы
levelУровень аудита, может принимать значения ak_log_none, ak_log_standard и ak_log_maximum.
Возвращает
Функция всегда возвращает ak_error_ok (ноль).

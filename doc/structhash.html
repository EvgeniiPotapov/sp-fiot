Подробное описание

Класс, реализующий контекст алгоритима хеширования.

Класс предоставляет интерфейс для реализации различных бесключевых алгоритмов хеширования информации, построенных с использованием итеративных сжимающих отображений. В настоящее время с использованием класса hash реализованы следующие отечественные алгоритмы хеширования

  • Стрибог256,
  • Стрибог512,
  • ГОСТ Р 34.11-94 (в настоящее время стандарт выведен из обращения).

Подробное описание методов использования объектов класса hash содержится в разделе Алгоритмы бесключевого хеширования.

Примеры:
example-internal-compress01.c, example-internal-hash01.c, example-internal-hash02.c и example-internal-hash03.c.

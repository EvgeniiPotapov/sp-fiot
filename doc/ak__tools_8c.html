)

Конвертация строки шестнадцатеричных символов в массив данных.

Функция преобразует строку символов, содержащую последовательность шестнадцатеричных цифр, в массив данных.

Аргументы
hexstrСтрока символов.
ptrУказатель на область памяти (массив), в которую будут размещаться данные.
sizeМаксимальный размер памяти (в байтах), которая может быть помещена в массив. Если исходная строка требует больший размер, то возбуждается ошибка.
reverseПоследовательность считывания байт в память. Если reverse равно ak_false то первые байты строки (с младшими индексами) помещаются в младшие адреса, а старшие байты - в старшие адреса памяти. Если reverse равно ak_true, то производится разворот, то есть обратное преобразование при котором элементы строки со старшиси номерами помещаются в младшие разряды памяти (такое представление используется при считывании больших целых чисел).
Возвращает
В случае успеха возвращается ноль. В противном случае, в частности, когда длина строки превышает размер массива, возвращается код ошибки.

char* ak_ptr_to_hexstr ( const ak_pointer  ptr,
const size_t  ptr_size,
const ak_bool  reverse 
)

Создание строки символов, содержащей значение заданной области памяти.

Функция рассматривает область памяти, на которую указывает указатель ptr, как массив последовательно записанных байт фиксированной длины, создает в оперативной памяти строку и последовательно выводит в нее значения, хранящиеся в заданной области памяти. Значения выводятся в шестнадцатеричной системе счисления.

Пример использования.

1 ak_uint8 data[5] = { 1, 2, 3, 4, 5 };
2 ak_uint8 *str = ak_ptr_to_hexstr( data, 5, ak_false );
3 printf("%s\n", str );
4 free(str);
Аргументы
ptrУказатель на область памяти
ptr_sizeРазмер области памяти (в байтах)
reverseПоследовательность вывода байт в строку. Если reverse равно ak_false, то байты выводятся начиная с младшего к старшему. Если reverse равно ak_true, то байты выводятся начиная от старшего к младшему (такой способ вывода принят при стандартном выводе чисел: сначала старшие разряды, потом младшие).
Возвращает
Функция возвращает указатель на созданную строку, которая должна быть позднее удалена пользователем с помощью вызова функции free(). В случае ошибки конвертации возвращается NULL. Код ошибки может быть получен с помощью вызова функции ak_error_get_code()
Примеры:
example-bckey-internal.c, example-hash.c, example-internal-compress01.c, example-random-system.c и example-randomize.c.

Преобразование области памяти в символьное представление.

Функция рассматривает область памяти, на которую указывает указатель ptr, как массив последовательно записанных байт фиксированной длины. Символьная (шестнадцатеричная) форма записи массива ptr помещается в заранее выделенный массив out. Если длины недостаточно, то возбуждается ошибка.

Пример использования.

1 ak_uint8 data[5] = { 1, 2, 3, 4, 5 };
2 ak_uint8 data_out[12];
3 
4 if( ak_ptr_to_hexstr_static( data, 5, data_out, 12, ak_false ) == ak_error_ok )
5  printf("%s\n", data_out );
Аргументы
ptrУказатель на область памяти
ptr_sizeРазмер области памяти (в байтах)
outУказатель на область памяти, в которую записывается символьное представление данных
out_sizeРазмер области памяти (в байтах); должен быть не менее, чем величина 1 + 2*ptr_size.
reverseПоследовательность вывода байт в строку. Если reverse равно ak_false, то байты выводятся начиная с младшего к старшему. Если reverse равно ak_true, то байты выводятся начиная от старшего к младшему (такой способ вывода принят при стандартном выводе чисел: сначала старшие разряды, потом младшие).
Возвращает
Если преобразование прошло успешно, возвращается ak_error_ok. В противном случае возвращается код ошибки.
Примеры:
example-internal-hash02.c, example-internal-hash03.c и example-mpzn-convert.c.

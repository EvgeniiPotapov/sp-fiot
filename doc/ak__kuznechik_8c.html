int ak_bckey_create_kuznechik ( ak_bckey  bkey)

Инициализация контекста секретного ключа алгоритма блочного шифрования Кузнечик.

Функция инициализируете контекст ключа алгоритма блочного шифрования Кузнечик. После инициализации устанавливаются обработчики (функции класса). Однако само значение ключу не присваивается - поле bkey->key остается неопределенным.

Аргументы
bkeyКонтекст секретного ключа алгоритма блочного шифрования.
Возвращает
Функция возвращает код ошибки. В случаее успеха возвращается ak_error_ok.

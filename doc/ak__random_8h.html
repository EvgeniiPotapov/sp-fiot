const char *  filename

Cоздание генератора, считывающего случайные значения из заданного файла.

Данный генератор связывается с заданным файлом и возвращает содержащиеся в нем значения в качестве случайных чисел. Если данные в файле заканчиваются, то считывание начинается с начала файла.

Основное назначение данного генератора - считывание данных из файловых устройств, таких как /dev/randon или /dev/urandom.

Аргументы
generatorКонтекст создаваемого генератора.
filenameИмя файла.
Возвращает
В случае успеха, функция возвращает ak_error_ok. В противном случае возвращается код ошибки.

ak_uint64 ak_random_value ( void  )

Функция генерации случайного 64-х битного целого числа.

Функция использует для генерации случайного значения текущее время и прочие параметры. Несмотря на случайность вырабатываемого значения, функция не должна использоваться для генерации значений, для которых требуется криптографическая случайность. Это связано с достаточно прогнозируемым изменением значений функции при многократных повторных вызовах.

Основная задача данной функции - инициализаци программного генератора каким-либо знаением, в случае, когда пользователь не инициализирует программный генератор самостоятельно.

Возвращает
Функция возвращает случайное число размером 8 байт (64 бита).

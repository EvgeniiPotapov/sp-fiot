Сложение двух вычетов

Функция реализует операцию сложения двух вычетов кольца , то есть реализует операцию . В качестве результата функции возвращается знак переноса r, принимающий значение либо 0, либо 1. Знак переноса позволяет интерпретировать сложение вычетов как сложение целых чисел, то есть записать точное равенство .

Допускается использовать в качестве аргумента z один из аргументов x или y.

Для максимальной эффективности вычислений функция не проверяет допустимые значения параметров.

Аргументы
zВычет, в который помещается результат
xВычет (левое слагаемое)
yВычет (правое слагаемое)
sizeРазмер вычетов в машинных словах - значение, задаваемое константой ak_mpzn256_size или ak_mpzn512_size
Возвращает
Функция возвращает значение знака переноса.

Сложение двух вычетов в представлении Монтгомери.

Функция складывает два вычета x и y по модулю p, после чего приводит полученную сумму по модулю p, то есть вычисляет значение сравнения . Результат помещается в переменную z. Указатель на z может совпадать с одним из указателей на слагаемые.

Удвоение на двойку в представлении Монтгомери.

Функция умножает вычет x на 2, после чего приводит полученную сумму по модулю p, то есть вычисляет значение сравнения . Умножение производится путем сдвига вычета x на 1 разряд влево и последующего вычитания модуля p.

Умножение двух вычетов как целых чисел

Функция реализует операцию умножения двух вычетов как двух целых чисел, то есть для вычисляется значение для которого в точности выполнено равенство .

Допускается использовать в качестве аргумента z один из аргументов x или y. Однако надо обязательно учитывать тот факт, что результат z занимает в два раза больше места чем x или y.

Для максимальной эффективности вычислений функция не проверяет допустимые значения параметров.

Аргументы
zВычет, в который помещается результат. Должен иметь длину в два раза большую, чем длины вычетов x и y.
xВычет (левый множитель)
yВычет (равый множитель)
sizeРазмер вычетов x, y в машинных словах - значение, задаваемое константой ak_mpzn256_size или ak_mpzn512_size.
Возвращает
Функция не возвращает значение.

dll_export const char* ak_libakrypt_get_mode_str ( ak_oid_mode  mode)

Получения символьного описания режима применения криптографического механизма.

Аргументы
modeрежим использованиея криптографического механизма
Возвращает
Функция возвращает указатель на строку, сожержащую описание типа криптографического механизма. Если значение engine неверно, то возбуждается ошибка и возвращается указатель на null-строку
